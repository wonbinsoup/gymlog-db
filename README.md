# GymLog - Workout Tracking Database

A relational database system for tracking gym workouts, exercises, sets, and personal records.

## Files
- `requirements.pdf` - Problem requirements and business rules
- `Blank diagram (3).png` - UML Class Diagram
- `erd.png` - Entity Relationship Diagram
- `Definition.pdf` - Relational schema with BCNF proof
- `gymlog_schema.sql` - SQL CREATE statements
- `gymlog_data.sql` - Test data INSERT statements
- `queries.sql` - SQL queries
- `db_screenshot.png` - DB format screenshot

## Database
Built with SQLite3. Contains 7 tables: User, Exercise, MuscleGroup, ExerciseMuscle, WorkoutSession, WorkoutSet, PersonalRecord.
